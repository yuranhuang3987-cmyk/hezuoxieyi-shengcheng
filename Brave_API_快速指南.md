## 📋 步骤

### 1️⃣ 访问 Brave API 网站

**打开浏览器，访问**：
```
https://api-dashboard.search.brave.com/app/dashboard
```

---

### 2️⃣ 登录或注册

**如果已登录**：
- 直接看到 Dashboard

**如果未登录**：
- 点击 "Sign Up" 或 "Log In"
- 可以用 Google/GitHub/Microsoft 账号快速登录

---

### 3️⃣ 获取 API Key

**在 Dashboard 中**：
1. 找到 **"API Keys"** 或 **"Subscriptions"**
2. 点击 **"Generate New Key"** 或 **"Get Started"**
3. 如果需要订阅，选择 **"Free"** 套餐
4. 复制生成的 API Key

**API Key 格式**：
```
BSA...（一长串字符）
```

**⚠️ 重要**：
- 只显示一次，务必保存
- 免费额度：2000 次/月
- 完全够用

---

**通过飞书私聊发送**：

