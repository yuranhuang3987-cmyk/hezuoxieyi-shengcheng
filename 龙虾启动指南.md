**最后更新**：2026-02-27

## 🚀 每天启动步骤

### 1️⃣ 打开 WSL2
```bash
# Windows 上打开 Ubuntu 终端
# 或者在 PowerShell/CMD 中输入：
wsl
```

### 2️⃣ 启动 OpenClaw Gateway
```bash
# 在 WSL2 中执行：
openclaw gateway
```

**成功标志：**
```
Gateway: bind=loopback (127.0.0.1), port=18789
Listening: 127.0.0.1:18789
```

### 3️⃣ 打开 Web 界面
```
浏览器访问：http://localhost:18789
```

就能看到聊天界面了！

---

## 💡 常用命令

### 检查状态
```bash
openclaw gateway status
```

### 后台运行
```bash
openclaw gateway &
```

### 重启 Gateway
```bash
openclaw gateway stop
openclaw gateway
```

### 查看代理状态
```bash
proxy-status  # 显示当前 IP（应该显示 VPN IP）
```

## 🔧 故障排查

### 问题 1：Gateway 启动失败
```bash
# 检查端口占用
lsof -i :18789

# 强制杀掉旧进程
kill -9 <PID>

# 重新启动
openclaw gateway
```

### 问题 2：代理不生效
```bash
# 重新加载配置
source ~/.bashrc

# 测试代理
curl https://api.ipify.org
```

### 问题 3：搜索功能不可用
```bash
# 测试搜索
export http_proxy="http://172.29.160.1:21882"
~/.openclaw/skills/freeUnlimited-websearch/search.py "test"
```
